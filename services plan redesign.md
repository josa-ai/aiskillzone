# Services Page Redesign Plan

We’ll keep the current structure and keep the “How we solve it” section card-based, but make the page feel more art-directed, fuller, and more premium through layout, typography, and surface styling.

## 1. Strengthen The White Sections

- Widen the content containers on desktop so the page feels less underfilled.
- Reduce some of the oversized vertical gaps between sections to tighten the rhythm.
- Introduce alternating section backgrounds: white, very light lavender, and subtle gradient haze so the eye keeps moving.
- Add more visual anchors inside each section so the space feels intentional, not empty.

## 2. Redesign The Intro Content Blocks

- Turn “The problem” into a more designed editorial block with a stronger heading, short supporting copy, and a visual accent such as a colored left rule, soft panel background, or oversized section label.
- Turn “What’s it costing you?” into a higher-impact card cluster instead of one quiet box.
- Keep the content concise, but present it with clearer hierarchy:
- stronger heading
- highlighted key phrases
- short bullet-style pain points
- optional small icon or metric callouts

## 3. Refine “How We Solve It” Without Changing The Structure

- Keep the image cards and the breakout layout.
- Redesign each card so they feel more polished and less like simple thumbnails:
- stronger border/shadow treatment
- cleaner caption styling
- consistent image ratios
- subtle hover lift
- slightly varied card heights or stagger for a more curated look
- Make the section itself more prominent by increasing the scale of the cards and giving the grid more breathing room horizontally, not just vertically.
- Add a stronger intro above the grid so the section feels like a showcase, not just a gallery.

## 4. Make The FAQ Feel Designed

- Wrap the FAQ in a larger branded container instead of letting it float on white.
- Use a soft tinted background, rounded corners, and a more premium accordion style.
- Add a short supporting line above the questions so it feels like part of the story, not an afterthought.

## 5. Improve The Visual Rhythm Before The Final CTA

- Create one stronger transition from the white-content area into the purple CTA banner.
- This can be done with a gradient divider, angled section edge, glow, or a subtle background shift so the CTA feels earned and dramatic.

## 6. Upgrade The Related Services Section

- Keep the cards, but make them feel like premium recommendations rather than generic footer content.
- Increase card contrast, spacing, hover behavior, and title styling.
- Consider adding a short intro line above the cards to position them as next-step offers.

## 7. Typography And Polish Pass

- Increase heading scale in the white sections.
- Tighten paragraph widths for readability while widening the overall section footprint.
- Use stronger contrast between headline, body, and supporting labels.
- Add a few subtle decorative details:
- section numbers
- eyebrow labels
- gradient accents
- soft shadows
- faint background texture

## Implementation Priority

1. Container widths and spacing rhythm
2. “The problem” and “What’s it costing you?” redesign
3. Card styling refresh for “How we solve it”
4. FAQ container redesign
5. Related services polish
6. Typography and visual detail pass

## Guardrails

- No new “What Your Brand Could Look Like” section
- No brand transformation snapshot
- No major structural change to “How we solve it”
- Keep the card-based information layout, but elevate the styling so it feels more impressive to clients
